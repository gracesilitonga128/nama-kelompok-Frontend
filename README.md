penjelasan struktur folder (yang dalam src ya ges)

Component = tau lah ya komponen
config = file configurasi jika ingin menggunakan environment react tertentu seperti babel, webpack, dsb (contoh file = config.js)
context = isinya file context API jadinya dapat mengatur props berbagai component sekaligus
Hooks = tau lah
services = folder isinya file untuk menangani api
utils = isinya file untuk membantu kek untuk manipulasi string atau format tanggal
styles = file css
assets = gambar / font, lebih baik pake google font
